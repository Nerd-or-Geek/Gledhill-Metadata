Quick start

Use this path for your first successful batch in under five minutes.

1) Import photos

Open the app and click Pick Photos, or drag files directly into the main view.

2) Select output rules

Choose a metadata template and naming convention that fits the shoot.

3) Review previews

Check fields and rename output before applying changes to your files.

4) Apply and verify

Commit changes, then spot-check a few files to confirm the expected result.

Tip: Keep one clean “golden template” for recurring workflows (events, products, archives).

Metadata templates

Templates standardize metadata entry and reduce repetitive editing across shoots.

  1. Create a template with your most-used fields.
  2. Name it clearly by workflow (e.g., Wedding-Editorial).
  3. Apply the template to selected photos in one action.
  4. Save revisions when standards change.
Recommended field order: Title → Description → Keywords → Headline → Creator → Copyright.

Renaming files safely

Troubleshooting

Photo picker does not open on macOS

Check macOS permissions in System Settings and ensure the app has file access permission.

Some metadata appears to remain after clearing

Certain embedded tags can be preserved for image integrity. Include exact tag names in bug reports.

Window size feels wrong on launch

Resize once and close normally; the app restores the last window dimensions on next launch.

Report better bugs, get faster fixes

Open Settings in the app and use Copy Bug Details before filing an issue.

  • What you expected
  • What happened instead
  • OS + app version
  • Sample filename or metadata key
